# 2 词法与构词法

## 2.1 结构词
- 结构词只有四种形态：
    1. 「V」：例如「.a」「.e」「.i」「.o」「.u」
    2. 「CV」：例如「ba」「ce」「di」「fo」「gu」
    3. 「V(')V」：例如「.au」「.ei」「.ia」「.o'u」「.u'e」
    4. 「CV(')V」：例如「ki'a」「pei」「mi'o」「coi」「cu'u」

    其中「C」表示辅音，「V」表示元音。任意两个连续的结构词都可以连写

### 2.1.2 代词
1. 「KOhA」类：可以替代项的结构词，分为几组
    1. 「KOhA1」类：逻辑存在量化项代词
        - 「da」：第一个「某人/某物」
        - 「de」：第二个「某人/某物」
        - 「di」：第三个「某人/某物」
    2. 「KOhA2」类：对语句的指代
        - 「da'u」：「很久之前说过的那句」，指代在较远的过去（某人）曾经说到的语句
        - 「da'e」：「很久之后要说的那句」，指代在较远的未来（某人）将会说到的语句
        - 「de'u」：「之前说过的那句」，指代（某人）最近说到的语句
        - 「de'e」：「将要说到的那句」，指代（某人）将要说到的语句
        - 「di'u」：「上一句」，指代上一个句子
        - 「di'e」：「下一句」，指代下一个句子
        - 「dei」：「这一句」，指代（某人）当前说的语句
        - 「do'i」：「省略的那句」，指代一个省略的或未指明的句子
    3. 「KOhA3」类：对主体的指代
        - 「mi」：「我；我所代表的那方面」，由「mi'e」确定指代的对象
        - 「mi'a」：「我（说话者）和其他人（第三方），不包括你（听者）」
        - 「mi'o」：「我（说话者）和你（听者），不包括其他人（第三方）」
        - 「ma'a」：「我（说话者）和你（听者）和其他人（第三方）」
        - 「do」：「你（听者）」，由呼语定义具体指代
        - 「do'o」：「你（听者）和其他未明确说明的人」
        - 「ko」：「你（祈使性的）」，你（听者）去使我的描述变为现实
    4. 「KOhA3a」类：对主体的指代
        - 「mi'ai」：「我们」，我和至少另外的一个人
        - 「da'ai」：「除了我以外的其他人」，指代人而非物体
    5. 「KOhA4」类
        - 「ko'a」：由「goi」指派的第一个代词
        - 「ko'e」：由「goi」指派的第二个代词
        - 「ko'i」：由「goi」指派的第三个代词
        - 「ko'o」：由「goi」指派的第四个代词
        - 「ko'u」：由「goi」指派的第五个代词
        - 「fo'a」：由「goi」指派的第六个代词
        - 「fo'e」：由「goi」指派的第七个代词
        - 「fo'i」：由「goi」指派的第八个代词
        - 「fo'o」：由「goi」指派的第九个代词
        - 「fo'u」：由「goi」指派的第十个代词
        - 「vo'a」：反身代词，所在谓句的第一项
        - 「vo'e」：反身代词，所在谓句的第二项
        - 「vo'i」：反身代词，所在谓句的第三项
        - 「vo'o」：反身代词，所在谓句的第四项
        - 「vo'u」：反身代词，所在谓句的第五项
    6. 「KOhA5」类：对前文提到项（不包括「KOhA4」类的项）的指代，项的距离以词组的开始位置为度量标准
        - 「ri」：前文出现的最近项，往前数第一个项
        - 「ra」：前文出现的稍远项，往前数第二个项
        - 「ru」：前文出现的更远项，在所有正在被指代的项的前面
    7. 「KOhA6」类：对客体的指代
        - 「ti」：「这」，说者附近的东西或地方
        - 「ta」：「那」，听者附近的东西或地方
        - 「tu」：「那」，远离说者和听者的事物
    8. 「KOhA7」类：对项的指代
        - 「ke'a」：在关系从句中指代被修饰的项
        - 「ma」：「什么」，对项发起提问
        - 「zo'e」：未指明值，具有一个可以使句子为真的值

2. 「GOhA」类：可以替代谓语的结构词
    - 「du」：「等于」，等价谓词
    - 「go'a」：指代较近的一个谓句，通常不是往前数最近的两个
    - 「go'e」：指代往前数第二个谓句
    - 「go'i」：指代前一个谓句，「le go'i」指代上一谓句的第一项
    - 「go'o」：指代之后的一个谓句，通常是紧接着的下一个谓句
    - 「go'u」：指代一个很久以前说过的谓句
    - 「mo」：「怎么」，对谓语发起提问
3. 「BY2」类：首字母指代前文最近一个项，这个项的谓语部分的首字母与该首字母一致

    !!! note ""
        - la .alis. goi ko'a citka[^1] le xunre[^2] plise[^3] .i py. se nelci[^4] ko'a

            爱丽丝吃那个红苹果。它（那个苹果）被爱丽丝所喜爱。

2. 「JOI」类：非逻辑连词，将两个以上谓句或项按照语法功能连接或合并为单一的谓句或项
    - 「ce」：构造一个无序集合
    - 「ce'o」：构造一个有序序列
    - 「fa'u」：非混合有序分布关系
    - 「jo'e」：取并集
    - 「joi」：混合所连的两个项，使其成为一个整体
    - 「jo'u」：协同，不将所连的词混合
    - 「ju'e」：与……相似
    - 「ku'a」：取交集
    - 「pi'u」：叉乘/集合的笛卡尔积

### 2.1.5 项构建词
1. 冠词
    1. 「LA」类：专名冠词，使专名变成项的词语
        - 「la」：用于专名之前，表示被命名为对应专名的那个对象
        - 「lai」：相当于「lu'o la」的缩写，被命名为对应专名的集团
        - 「la'i」：被命名为对应专名的组
    2. 「LE」类：普通项冠词，使谓语变成项的词语，其描述或指定适合谓语第一位的事物
        - 「le」：用于谓语之前，表示被描述为对应谓语的对象（表示说者心中所想的对象并用其指称，尽管这种概括不一定准确）
        - 「lei」：相当于「lu'o le」的缩写，被描述为对应谓语的集团
        - 「le'i」：被描述为对应谓语的组
        - 「le'e」：按刻板印象将其描述为对应谓语的对象
        - 「lo」：用于谓语之前，表示某个真实存在的对象（表示适合该谓语第一位的对象）
        - 「loi」：相当于「lu'o lo」的缩写，真实存在的集团
        - 「lo'i」：真实存在的组
        - 「lo'e」：具有某种典型性的对象
    3. 「LI」类：数量词冠词，使结构词组成的数字变成项的词语，只含有「li」一个词语
    4. 「LAhE」类：指代型冠词，在无法直接表示指定项时用于指代某个或某些具体的项
        - 「la'e」：间接指代的指向物，将项所指向的事物作为项实际指代的事物
        - 「lu'a」：组成集合或群体的个体，将其他的描述类型转换为个体描述
        - 「lu'e」：「……的标志」，用代表项的标志或词语替代真实的项
        - 「lu'i」：「含有……作为成员的集合」，将其他描述类型转换为集合描述
        - 「lu'o」：「由……组成的群体」，将其他描述类型转换为群体描述
2. 引用词：引用一个文本块，通常不省略对应的终止词
    1. 「LU」类：错误引用，只含有「lu」一个词语。开始一个对有语法错误或与语境无关的文本引用
    2. 「LOhU」类：完整语法引用，只含有「lu'o」一个词语。开始一个对符合语法的文本引用
3. 其他项构建词
    - 「BE」类：用于向冠词修饰的谓语添加其他项
        1. 不使用「BAI」类或「FA」类时默认填充第二项
        2. 连续使用「be」是对新项的修饰
    - 「BEI」类：用于向冠词修饰的谓语添加第二及以后的项，不使用「FA」类时默认填充第二项
    - 「NU」类：谓词子句指示词，将完整的谓句变为谓语。如果前面接续冠词，则进一步转化为项
        - 「nu」：描述真实世界发生的事件
        - 「du'u」：$x_1$ 是心中所想的事件，用 $x_2$ 表达出来
        - 「su'u」：通用指示词，用以代替其他「NU」类结构词

### 2.1.6 项修饰词
1. 「GOI」类：关系标记词，相当于只有一项的关系从句
    1. 「goi」：项指派，类似于「以下简称……」，后接「KOhA4」类
    2. 限制性关系短语标记，后接一项
        - 「pe」：「与……相关」，表示最松散的相关性
        - 「po」：「特定于……」，表示属格，物理上的或法律上的拥有关系
        - 「po'e」：「属于……」，表示不可分割的拥有关系
        - 「po'u」：「即……」，表示同位语

        !!! note ""
            - le skari[^7] pe le plise

                苹果的颜色

            - le plise po mi 

                我的苹果

            - le mensi po'e la .alis.

                爱丽丝的妹妹

        特别地，当物主是代词或专名表示的项时，可以将「pe」与物主放在冠词与根词项之间，且「pe」可以被省略

        !!! note ""
            - le pe la .alis. mensi

                爱丽丝的妹妹

            - le do plise

                你的苹果

    3. 非限制性关系短语标记（有「顺便说一下」之意），后接一项
        - 「ne」：「与……相关」，与「pe」相对
        - 「no'u」：「即……」，与「po'u」相对

2. 「NOI」类：关系从句指示词
    - 「poi」：引导限制性关系从句，用以表示辨别性信息
    - 「noi」：引导非限制性关系从句，用以表示附加信息
    - 「voi」：引导非真实限制性关系从句

### 2.1.7 谓语修饰词
1. 谓词含义变换：调整谓词的原有语法结构以更改语序或含义，直接作用于谓词或者指定的项
    1. 「FA」类：位标签
        - 「fa」：根词的第一位
        - 「fe」：根词的第二位
        - 「fi」：根词的第三位
        - 「fo」：根词的第四位
        - 「fu」：根词的第五位
    2. 「SE」类：将谓句与部分结构词中对应的项换位
        - 「se」：将第一位与第二位换位
        - 「te」：将第一位与第三位换位
        - 「ve」：将第一位与第四位换位
        - 「xe」：将第一位与第五位换位
2. 谓词含义修饰：直接作用于谓词或者指定的项，可用于组建词组或合成词
    1. 「NAhE」类：谓句程度修饰
        - 「je'a」：肯定其后的谓语
        - 「je'ai」：对「NAhE」类结构词发起提问
        - 「na'e」：否定其后的谓语
        - 「no'e」：否定其后的谓语，并表示相对该谓语更加中立的立场
        - 「to'e」：否定其后的谓语，并表示与该谓语相反的意思
    2. 「NA」类：谓句字面义修饰，作用域为整个谓句
        - 「na」：表示对谓句的否定
        - 「ja'a」：表示对谓句的逻辑确认
    3. 「NAI」类：结构词字面义修饰
        - 「nai」：表示对结构词的否定
        - 「ja'ai」：表示对结构词的确认

### 2.1.8 事件修饰词
事件修饰词及其序列都可以放在谓词之前，使整个谓句进入相应时态/位态/状态

1. 时态修饰词：
    1. 「PU」类：表示时态关系，用于对事件的发生时间进行相对定位。通常后接一项
        - 「pu」：「在……之前」，默认的过去时
        - 「ca」：「与……同时」，默认的现在时
        - 「ba」：「在……之后」，默认的将来时

        利用冠词的终止词「ku」，可以省略作为参照的事件

        !!! note ""
            - pu lenu do citka kei mi citka le plise

                在你吃东西之前，我吃了那个苹果。

            - caku mi citka le plise

                现在我吃了那个苹果。

    2. 「ZEhA」类：表示时态区间
        - 「ze'i」：「不长时间」，很短的时间长度
        - 「ze'a」：「稍长时间」，中等时间长度
        - 「ze'u」：「很久长时间」，很长的时间长度
        - 「ze'e」：「一直/永远」，整段时间
    3. 「ZI」类：时态距离，具体时间长短依据说话者主观判断决定
        - 「zi」：「距离不久」，很短的时间距离
        - 「za」：「距离稍久」，中等长度的时间距离
        - 「zu」：「距离很久」，很长的时间距离

2. 位态修饰词
    1. 「VA」类：表示位态距离，用于对事件的发生位置进行相对定位。通常后接一项
        - 「vi」：「这里」；在很近范围内
        - 「va」：「那里」；在中等范围内
        - 「vu」：「更远的那边」；在很远范围内

        利用终止词「ku」，可以以说者作为参照

        !!! note ""
            - vi lenu do citka kei mi citka le plise

                我在你吃东西的地方吃了那个苹果。

            - vaku mi citka le plise

                我在那里吃那个苹果。

    2. 「VEhA」类：表示位态区间
        - 「ve'i」：「极小区域」，一个极小的区域
        - 「ve'a」：「小区域」，一个小的区域
        - 「ve'u」：「大区域」，一个大的区域
        - 「ve'e」：「整个区域」，整个区域
    3. 「VIhA」类：表示位态区间维度
        - 「vi'i」：「一维空间」，一条线
        - 「vi'a」：「二维空间」，平面区域
        - 「vi'u」：「三维空间」，三维空间区域
        - 「vi'e」：「四维空间」，时空区域
    4. 「FAhA」类：表示位置关系。通常可以放在「VA」类前，也可以直接后接一项
        - 「FAhA1」类
            - 「du'a」：「在……东边」
            - 「ne'u」：「在……南边」
            - 「vu'a」：「在……西边」
            - 「be'a」：「在……北边」
        - 「FAhA2」类
            - 「ga'u」：「上方」
            - 「ni'a」：「下方」
            - 「zu'a」：「左边」
            - 「ri'u」：「右边」
            - 「ca'u」：「前方」
            - 「ti'a」：「后方」
            - 「ru'u」：「环绕」
        - 「FAhA3」类
            - 「re'o」：「与……靠着」
            - 「te'e」：「至……边缘」
            - 「bu'u」：「在同一个位置」
            - 「ne'a」：「在……的旁边」
        - 「FAhA4」类
            - 「zo'i」：「从……向里」
            - 「ze'o」：「从……向外」
            - 「zo'a」：「相切/路过」
            - 「to'o」：「从……离开」
            - 「fa'a」：「向……而去」
    5. 「MOhI」类：「朝向……运动」，只含有「mo'i」一个词语，标记空间中的移动

3. 状态修饰词
    - 「ZAhO」类：事件区间特性
        - 「co'i」：发生时，「在……发生的瞬间」，适用于即时事件
        - 「co'u」：暂停时，「在……的结束时点（甚至事件并未全部完成）」，表示事件的停止
        - 「co'a」：开始，「在……开始的时间点」
        - 「pu'o」：之前，「在……以前；直到……」
        - 「ca'o」：正在，「在……的时候」，继续体
        - 「ba'o」：之后，「在……之后的时间段」，完成时
        - 「mo'u」：完成了，「在……的自然结束点」，完成时
        - 「de'a」：中断，表示暂停的时间特性或者一个过程的可恢复的中断
        - 「di'a」：恢复，表示恢复一个暂停的过程的事件特性
        - 「xa'o」：太早，指代事件在自然的开始点之前的部分，过早的在……之前开始了
        - 「za'o」：太晚，持续了过长时间，延续到了事件……的自然结束时间的后面，过完成式

### 2.1.13 单例类
1. 「BO」类：接续在「BAI」类后，使「BAI」后不接续一项而是一个谓句

3. 「SOI」类：附在在谓句之后，后接两个谓句中的项，表示两项交换后原句也成立。第二项省略时默认为「soi」前的最近项

4. 「ZIhE」类：用以连接同时修饰同一项的多个关系从句

## 2.3 专名
- 专名是以辅音结尾的词语，而结构词与谓词均以元音结尾
    1. 专名尾部需附上一个句点以确认词语的边界
    2. 专名可以通过转写其他语言中的名字得到或通过改造逻辑语中的结构词与谓词得到

        !!! note "专名的例子"
            1. 英语中「Mary」转写为逻辑语中的「meris.」或「mer.」
            2. 英语中「Ahmet」转写为逻辑语中的「.axmet.」
            3. 法语中「Pierre」转写为逻辑语中的「pi,ER.」或「pi.ER.」
            4. 英语中「Green」对应逻辑语中「crino」，可写为「crinos.」或「crin.」

[^1]: citka：$x_1$ 吃/消化 $x_2$
[^2]: xunre：$x_1$ 是红色的/红润的
[^3]: plise：$x_1$ 是苹果，品种是 $x_2$
[^4]: nelci：$x_1$ 喜欢/热衷于 $x_2$
[^5]: kukte：$x_1$ 是美味的/令人愉快的，对于体验者/感官 $x_2$ 来说
[^6]: mensi：$x_1$ 是 $x_2$ 的（不一定是生物学意义上的）姐妹，以 $x_3$ 关系联结
[^7]: skari：$x_1$ 的颜色是 $x_2$，用 $x_3$ 看见/感知，在 $x_4$ 的条件下
