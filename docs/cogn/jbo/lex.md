# 2 词法与构词法

## 2.1 结构词
- 结构词只有四种形态：
    1. 「V」：例如「.a」「.e」「.i」「.o」「.u」
    2. 「CV」：例如「ba」「ce」「di」「fo」「gu」
    3. 「VV」：例如「.au」「.ei」「.ia」「o'u」「u'e」
    4. 「CVV」：例如「ki'a」「pei」「mi'o」「coi」「cu'u」

    其中「C」表示辅音，「V」表示元音

### 2.1.1 冠词
1. 「LA」类：使专名变成项的词语
    - 「la」：用于专名之前，表示被命名为对应专名的那个对象
    - 「lai」：相当于「lu'o la」的缩写，被命名为对应专名的集团
    - 「la'i」：被命名为对应专名的组
2. 「LE」类：使谓语变成项的词语，其描述或指定适合谓语第一位的事物
    - 「le」：用于根词之前，表示被描述为对应根词的对象（表示说者心中所想的对象并用其指称，尽管这种概括不一定准确）
    - 「lei」：相当于「lu'o le」的缩写，被描述为对应根词的集团
    - 「le'i」：被描述为对应根词的组
    - 「le'e」：按刻板印象将其描述为对应根词的对象
    - 「lo」：用于根词之前，表示某个真实存在的对象
    - 「loi」：相当于「lu'o lo」的缩写，真实存在的集团
    - 「lo'i」：真实存在的组
    - 「lo'e」：具有某种典型性的对象
3. 「LI」类：使结构词组成的数字变成项的词语，只含有「li」一个词语

### 2.1.2 代词
1. 「KOhA」类：可以替代项的结构词，分为几组
    1. 「KOhA1」类：量化存在代词
        - 「da」：第一个「某人/某物」
        - 「de」：第二个「某人/某物」
        - 「di」：第三个「某人/某物」
    2. 「KOhA3」类：对主体的指代
        - 「mi」：「我」，即说者
        - 「do」：「你」，即听者
        - 「ko」：「你」，即听者，用于命令句中
    3. 「KOhA4」类：由「goi」指派的代词
        - 「ko'a/ko'e/ko'i/ko'o/ko'u」：第一到第五个代词
        - 「fo'a/fo'e/fo'i/fo'o/fo'u」：第六到第十个代词
    4. 「KOhA5」类：对前文提到项（不包括「KOhA4」类的项）的指代，项的距离以词组的开始位置为度量标准
        - 「ri」：前文出现的最近项

            !!! note ""
                - la .alis. ca citka[^1] le ri plise[^2]

                    爱丽丝正在吃她自己的苹果。

        - 「ra」：前文出现的稍远项（非量化）
        - 「ru」：前文出现的更远项（非量化）

    5. 「KOhA6」类：对客体的指代
        - 「ti」：这，说者附近的东西或地方
        - 「ta」：那，听者附近的东西或地方
        - 「tu」：那，远离说者和听者的事物
    6. 「KOhA7」类：对项的指代
        - 「ma」：「什么」，对项发起提问，用于疑问句中
        - 「zo'e」：未指定值，表示该项在谓句中不重要

2. 「GOhA」类：可以替代谓语的结构词
    - 「go'i」：表示上一谓句的意义，「le go'i」指代上一谓句的第一项
    - 「mo」：「是什么/做什么」，对谓语发起提问，用于疑问句中
3. 首字母：首字母指代前文最近一个项，这个项的谓语部分的首字母与该首字母一致

    !!! note ""
        - la .alis. goi ko'a citka le plise .i py. se nelci[^3] ko'a

            爱丽丝吃那个苹果。它（那个苹果）被爱丽丝所喜爱。

### 2.1.3 数词
1. 「PA」类：表示数或位，以及与数相关的结构
    1. 确数（「PA1」类与「PA2」类）：表示数字时，直接将各数位连接。例如用「panorevo」表示「$1024$」

        | 数词  | 释义  | 数词  | 释义  | 数词  |    释义    | 数词  |    释义    |
        | :---: | :---: | :---: | :---: | :---: | :--------: | :---: | :--------: |
        |  no   |  $0$  |  vo   |  $4$  |  bi   |    $8$     |  gai  | $\text{C}$ |

        |  ci   |  $3$  |  ze   |  $7$  |  fei  | $\text{B}$ |  vai  | $\text{F}$ |

        </div>

    2. 与数相关的结构（「PA3」类）
        1. 「ki'o」：充当数字中的分隔符，当「ki'o」后的数字被省略时，相当于省略了一组「nonono」

            !!! note ""
                - pa ki'o norevo

                    $1,024$

                - so ki'o

                    $9,000$

                - pa ki'o ki'o

                    $1,000,000$

        2. 「pi」：表示小数点，例如用「cipipavo」表示「$3.14$」

            「pi'e」：表示数字的分隔符，通常在两边空格，常用于时间或日期

        3. 「ni'u」：表示负号，例如用「ni'u pano」表示「$-10$」

            「fu'u」：表示分号，例如用「refi'uci」表示「$2/3$」，一般以标识整体的冠词或「lu'o」接续

    3. 概数（「PA4」类）

        <div class="text-table">

        | 数词  |   释义   | 数词  |  释义  | 数词  | 释义  |
        | :---: | :------: | :---: | :----: | :---: | :---: |
        |  ro   |   每个   | so'e  | 大多数 | du'e  | 太多  |
        | so'a  | 几乎所有 | so'i  |  许多  | su'e  | 之多  |
        | so'u  |   少许   | so'o  |  若干  | su'o  | 至少  |

        </div>

        此外，「xo」表示「多少」，用于疑问句中对数词发起提问

2. 数词的用法
    1. 数词可置于「le」的前后表示特指

        !!! note ""
            - ci le plise

                那些苹果中的三个

            - le ci plise

                那三个苹果

            - re le ci plise

                那三个苹果中的两个

    2. 数词可置于「lo」的前后表示泛指。当置于「lo」后时，将指定泛指对象的总数

        !!! note ""
            - ci lo plise

                三个苹果（世界上所有苹果中的三个）

            - lo ci plise

                世界上仅存的三个苹果

            - re lo ci mensi[^4]

                我三个妹妹中的两个（说者有且仅有三个妹妹）

    3. 可在数词前附加「lu'o」表示一个整体，也可以使用「loi」或「lei」

        !!! note ""
            - mi citka ci lo plise

                mi citka ci lu'o lo plise

                我吃了三个苹果（每个苹果都被吃了，但三个苹果之间不一定存在联系）

                我吃了三个苹果（说者吃了作为一个整体的「三个苹果」，但并不一定每个苹果都被吃了）

            - pafi'ure lo plise

                pafi'ure loi plise

                半个苹果

                全世界所有苹果的一半

### 2.1.4 时态词与位态词
1. 任何关于时间的结构词或结构词序列都可以放在谓词之前，使整个谓句进入相应时态

    !!! note ""
        - mi puzize'i citka le plise

            我刚刚一下子就吃了那个苹果。

    1. 「PU」类：表示时态关系，用于对事件的发生时间进行相对定位。通常后接一项
        - 「pu」：意为「在……之前」
        - 「ba」：意为「在……之后」
        - 「ca」：意为「与……同时」

        利用终止词「ku」（可以连写），可以省略作为参照的事件

        !!! note ""
            - pu lenu do citka kei mi citka le plise

                在你吃东西之前，我吃了那个苹果。

            - caku mi citka le plise

                现在我吃了那个苹果。

    2. 「ZEhA」类：表示持续时间长短
        - 「ze'i」：很短时间
        - 「ze'a」：中等时间
        - 「ze'u」：很长时间
    3. 「ZI」类：时态距离，附在「PU」类结构词后，表示距离现在的时间长短
        - 「zi」：很短时间
        - 「za」：中等时间
        - 「zu」：很长时间

        具体时间长短依据说话者主观判断决定，例如「puzi」表示「在……的很短时间之前」

2. 任何关于位置的结构词或结构词序列都可以放在谓词之前，使整个谓句进入相应位态

    !!! note ""
        - mi vi citka

            我在这里吃东西。

    1. 「VA」类：表示位态关系，用于对事件的发生位置进行相对定位。通常后接一项
        - 「vi」：这里；在很近范围内
        - 「va」：那里；在中等范围内
        - 「vu」：更远的那边；在很远范围内

        利用终止词「ku」（可以连写），可以以说者作为参照

        !!! note ""
            - vi lenu do citka kei mi citka le plise

                我在你吃东西的地方吃了那个苹果。

            - vaku mi citka le plise

                我在那里吃那个苹果。

    2. 「FAhA」类：表示位置关系。通常可以放在「VA」类前，也可以直接后接一项
        - 「bu'u」：就在……的位置
        - 「ne'a」：靠近……
        - 「to'o」：远离……
        - 「ne'i」：在……的内部
        - 「ri'u」：在……的右边
        - 「zu'a」：在……的左边
        - 「fa'a」：在……的方向上
    3. 「MOhI」类：只含有「mo'i」一个词语，表示运动状态。例如「mo'ine'i」表示向内运动

3. 时态词与位态词可以一起使用，例如用「puzuvuku」表示「很久以前在一个遥远的地方」

### 2.1.5 终止词
- 终止词：用以标志一段语言结构的结束，避免出现句法二义性
    1. 「BOI」类：只含有「boi」一个词语，是数词的终止词
    2. 「LOhO」类：只含有「lo'o」一个词语，是「LI」类对应的终止词
    3. 「KEI」类：只含有「kei」一个词语，是「nu」引导谓词子句的终止词
    4. 「KU」类：只含有「ku」一个词语，是「LA」类或「LE」类对应的终止词
    5. 「VAU」类：只含有「vau」一个词语，是项的终止词，表示该谓句的所有项均已结束

### 2.1.6 项标签
- 「BAI」类：将额外的项嵌入到根词中
    - 「de'i」：「detri[^5]」的第一位，表示发生的日期
    - 「ti'u」：「tcika[^6]」的第一位，表示发生的时刻

### 2.1.7 呼语
1. 「COI」类：表示正在与某人说话（或正在说话的人）的短语
    - 「coi」：问候语，表示「你好」
    - 「mi'e」：后接介绍说者的词语，表示「我是……」
2. 「UI」类：态度指示词，是特殊的呼语。表示说话者的情绪状态、知识来源或当前话语阶段的助词

    2. 态度指示词之间可以组合，例如用「.u'e.ii」表示「疑惑且害怕」

### 2.1.8 其他词类
1. 「FA」类：即位标记
    - 「fa」：根词的第一位
    - 「fe」：根词的第二位
    - 「fi」：根词的第三位
    - 「fo」：根词的第四位
    - 「fu」：根词的第五位
2. 「GOI」类：限制性关系短语标记词
    1. 属格
        - 「pe」：最松散的属格，表示「与……有关」
        - 「po」：正常属格，表示「特定于……」
        - 「po'e」：表示不可剥夺的财产，表示「属于……」

        !!! note ""
            - le skari[^7] pe le plise

                苹果的颜色

            - le plise po mi 

                我的苹果

            - le mensi po'e la .alis.

                爱丽丝的妹妹

        特别地，当物主是代词或专名表示的项时，可以将「pe」与物主放在冠词与根词项之间，且「pe」可以被省略

        !!! note ""
            - le pe la .alis. mensi

                爱丽丝的妹妹

            - le do plise

                你的苹果

    2. 「goi」：类似于「以下简称……」，后接「KOhA4」类

3. 「LAhE」类
    - 「la'e」：一个项的代指
    - 「lu'a」：用在项之前，表示该项相互分离
    - 「lu'o」：用在项之前，表示该项是一个整体
4. 「NA」类
    - 「na」：用在谓句的谓语之前（但在「cu」之后），表示否定整个谓句

        !!! note ""
            - mi na nelci ro plise

                「我喜欢所有的苹果」这件事不是真的。

    - 「ja'a」：表示肯定整个谓句

5. 「NAhE」类：用在谓句的谓语之前
    - 「na'e」：否定其后的谓语
    - 「to'e」：否定其后的谓语，并表示相对该谓语更加中立的立场
    - 「no'e」：否定其后的谓语，并表示与该谓语相反的意思

    !!! note ""
        - mi na'e nelci ro plise

            我没有喜欢所有的苹果。

        - mi to'e nelci ro plise

            我对所有的苹果感觉一般。

        - mi to'e nelci ro plise

            我讨厌所有的苹果。

6. 「NU」类：将某种成分转化为另一种成分
    - 「nu」：谓词子句指示词，将完整的谓句变为谓语，如果前面接续冠词，则进一步转化为项
7. 「SE」类：将谓句中对应的项换位
    - 「se」：将谓句中的第一位与第二位换位
    - 「te」：将谓句中的第一位与第三位换位
    - 「ve」：将谓句中的第一位与第四位换位
    - 「xe」：将谓句中的第一位与第五位换位
8.  只包含一个词语的结构词类
    - 「CU」类：使用可删除标记「cu」可以用于分隔两个相邻谓语，使其不成为一个词组

        !!! note ""
            - le mensi cu citka

                我的妹妹在吃东西。

            - le mensi klama

                那个像我妹妹一样吃东西的人

    - 「I」类：分隔两个句子，所有句子均默认以「.i」开头
    - 「NAI」类：将「nai」附在结构词后表示否定

## 2.3 专名
- 专名是以辅音结尾的词语，而结构词与谓词均以元音结尾
    1. 专名尾部需附上一个句点以确认词语的边界
    2. 专名可以通过转写其他语言中的名字得到或通过改造逻辑语中的结构词与谓词得到

        !!! note "专名的例子"
            1. 英语中「Mary」转写为逻辑语中的「meris.」或「mer.」
            2. 英语中「Ahmet」转写为逻辑语中的「.axmet.」
            3. 法语中「Pierre」转写为逻辑语中的「pi,ER.」或「pi.ER.」
            4. 英语中「Green」对应逻辑语中「crino」，可写为「crinos.」或「crin.」

[^1]: citka：$x_1$ 吃/消化 $x_2$
[^2]: plise：$x_1$ 是苹果，品种是 $x_2$
[^3]: nelci：$x_1$ 喜欢/热衷于 $x_2$
[^4]: mensi：$x_1$ 是 $x_2$ 的（不一定是生物学意义上的）姐妹，以 $x_3$ 关系联结
[^5]: detri：$x_1$ 是 $x_2$ 的日期，位置是 $x_3$，按照历法 $x_4$
[^6]: tcika：$x_1$ 是 $x_2$ 的时刻，日期是 $x_3$，位置是 $x_4$，按照历法 $x_5$
[^7]: skari：$x_1$ 的颜色是 $x_2$，用 $x_3$ 看见/感知，在 $x_4$ 的条件下
