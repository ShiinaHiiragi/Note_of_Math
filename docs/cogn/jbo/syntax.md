# 3 句法与篇章法

## 3.1 结构词与句法
### 3.1.1 谓句
1. 逻辑语中的谓句：谓语与项
    1. 项是谓语意义对应的位，是词语所扮演的角色
    2. 谓语是表示项之间关系的词语，根词是谓语的重要组成部分
    3. 谓语可以被置于除句首外的任何位置（否则会被认为省略了第一位），通常在第一位与第二位之间

    !!! note "位结构的一般规律"
        重要位通常尽可能靠前，最少用到的位总是置于最后

        1. 第一位通常是是某人或某物，它要做什么或者是什么
        2. 如果某事或某物是被施加动作或接受影响的，那么其通常占据第二位
        3. 「到哪里去」通常总是在「从哪里来」之前

2. 位的变换
    1. 位的省略：当第一位或某一位及其后的位不重要时，可以直接将其省略
        1. 当某一位被省略，但其后的位无法省略时，使用代词「zo'e」显式省略
        2. 省略了所有项的谓句称为显式，表示无需任何其它细节的说明，通常用于说者向听者提示眼前有某种事物或即将发生某事

        !!! note ""
            - citka

                吃（相当于「zo'e citka zo'e」）。

    2. 位序标记：用「fa」「fe」「fi」「fo」「fu」分别指定前五个位

        !!! note ""
            - fe le plise cu citka fa mi

                那个苹果被我吃了。

    3. 换位：用「se」「te」「ve」「xe」换位以强调某一项

        !!! note ""
            - le plise cu se citka mi

                那个苹果被我吃了。

            - le citka cu citka le se citka

                吃东西的人吃了被吃的东西

3. 特殊句式
    1. 命令句：使用了代词「ko」的句子，表示要求听者做某事。在命令的句首使用了态度指示词「.e'o」的句子，表示礼貌的请求
    2. 疑问句
        1. 一般疑问句：在谓句的任何位置使用了态度指示词「xu」的句子，表示询问
            - 通过重复疑问句中的谓语，可以作为回复；若表示否定的回复，则在谓语前用「na」

                !!! note ""
                    - xu do nelci lo plise

                        nelci

                        na nelci

                        你喜欢苹果吗？

                        喜欢。

                        不喜欢。

            - 通过使用代词「go'i」，可以作为回复；若表示否定的回复，则在「go'i」前用「na」

                !!! note ""
                    - xu do nelci lo plise

                        go'i

                        na go'i

                        你喜欢苹果吗？

                        是的。

                        不是的。

        2. 特殊疑问句：通常使用对应的项或谓语回答

            - 用「ma」对项发起提问

                !!! note ""
                    - do citka ma

                        le plise

                        你要吃什么？

                        这个苹果。

            - 用「mo」对谓语发起提问

                !!! note ""
                    - do mo le plise

                        citka

                        你把那个苹果怎么了？

                        我把它吃了。

                    - la .alis. mo

                        mensi

                        爱丽丝是谁？

                        我的妹妹。

            - 用「xo」对数词发起提问

                !!! note ""
                    - xo le do mensi cu nelci do

                        re

                        你有多少个妹妹喜欢你？

                        两个。

### 3.1.2 子句与从句
1. 谓词子句：「NU」类引导的谓句，用以充当一项。该类可以通过「SE」类换位形成复合结构词，例如「sedu'u」

    !!! note ""
        - le la .alis. se pensi[^1] cu du'u ri nelci lo plise kei lu mi citka le plise li'u

            爱丽丝想着自己喜欢苹果，说「我吃那个苹果」（la .alis. 是属格用法）。

        - la .alis. cusku[^2] le sedu'u ri nelci lo plise

            爱丽丝说她喜欢苹果。

2. 关系从句：「NOI」类引导的谓句，用以修饰一项

    !!! note ""
        - le plise poi ke'a xunre ku'o se citka mi

            那个红苹果被我吃了。

## 3.2 根词与篇章法
### 3.2.1 句段
1. 谓句的分隔
    1. 使用「.i」分隔两个句子
    2. 「.i」 与「JA」类组成的复合结构词表示谓句间逻辑关系
2. 段落的分隔

### 3.2.2 时间
1. 时刻：用「tcika」描述时刻时，默认使用二十四小时制，「pi'e」分隔顺序依次为时、分、秒

    !!! note ""
        - ma tcika ti

            caku ma tcika

            li pano pi'e mu

            li pano pi'e ni'u mu

            li panopimu

            现在是几点？

            现在是几点？

            十点零五分。

            十点差五分。

            十点半。

2. 日期：用「detri」描述日期时，「pi'e」分隔顺序依次为日、月、年

    !!! note ""
        - li pano pi'e pare detri ti

            现在是十二月十日。

3. 用「nu」描述事件发生的时间，用「SE」类词换位。也可直接使用「ti'u」或「di'e」指定动作发生的时间

    !!! note ""
        - la daucac. tcika lenu mi citka

            十点钟就是我吃东西的时间。

        - lenu mi jbena[^3] cu se detri li pamu pi'e bi

            我出生的日期是八月十五日。

        - mi citka ti'u la daucac.

            我在十点钟吃东西。

        - mi jbena de'i li pamu pi'e bi

            我出生于八月十五日。

4. 小时、星期与月份
    1. 使用表示早晚的根词「clira[^4]」、「lerci[^5]」以及表示小时的根词「cacra[^6]」组成的专名表示十二小时制

        !!! note ""
            - li mucac.lir. tcika ti

                现在是凌晨五点。

    2. 使用表示星期的根词「djedi[^7]」组成的专名表示某星期

        </div>

        各地可使用其习惯的星期日，直到有一种通用的约定被共同接受

    3. 使用表示月份的根词「masti[^8]」组成的专名表示某星期

        <div class="text-table">

        |  专名   | 释义  |  专名   | 释义  |   专名   |  释义  |

        </div>

    4. 使用表示年份的根词「nanca[^9]」组成的专名表示某一年，例如用「la renonopanac.」表示二〇〇一年

### 3.2.3 因果
1. 用项标签表示因果关系：可以在项标签前附加「se」，将表示原因转变为表示结果
    - ki'u：表示原因，通常有一般性的人类行为规则和标准
    - mu'i：表示动机，通常出于个人的自由意志
    - ni'i：表示蕴含，表明逻辑必然性的结果
    - ri'a：表示物理原因，表明物理规律性的结果
2. 对原因进行提问：使用结构词「ma」

    !!! note ""
        - le plise cu xunre ri'a ma

            为什么这个苹果是红色的？

[^1]: pensi：$x_1$ 思考/反思 $x_2$
[^2]: cusku：$x_1$ 表达/说 $x_2$，听众是 $x_3$，通过媒介 $x_4$
[^3]: jbena：$x_1$ 为 $x_2$ 所生，日期是 $x_3$，地点是 $x_4$
[^4]: clira：$x_1$（事件/状态）较早，基于标准 $x_2$
[^5]: lerci：$x_1$（事件/状态）较晚，基于标准 $x_2$
[^6]: cacra：$x_1$（事件/状态）的持续时间是 $x_2$ 小时（默认为 $1$ 小时），基于标准 $x_3$
[^7]: djedi：$x_1$（事件/状态）的持续时间是 $x_2$ 天（默认为 $1$ 天），基于标准 $x_3$
[^8]: masti：$x_1$（事件/状态）的持续时间是 $x_2$ 月（默认为 $1$ 月），基于标准 $x_3$
[^9]: nanca：$x_1$（事件/状态）的持续时间是 $x_2$ 年（默认为 $1$ 年），基于标准 $x_3$
