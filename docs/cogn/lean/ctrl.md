# 2 控制结构

## 2.1 项
1. 类型 `α` 的实例 `x` 为项
2. 若函数 `f: α → β`，且有项 `t: α`，则 `f t` 为类型 `β` 的项
3. 若变元 `x: α` 且项 `t: β`，则函数 `fun x: α => t` 为类型 `α → β` 的项

    ```haskell
    #check Nat.zero         -- Nat
    #eval HAdd.hAdd 0 1     -- 1
    #check fun x: Nat => x  -- Nat → Nat
    ```
