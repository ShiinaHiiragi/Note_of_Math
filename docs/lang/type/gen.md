# 1 一般类型论

## 1.1 无类型 λ 演算
1. 构造原理
    1. 抽象：对于表达式 $M$ 与变量 $x$，可构造新的表达式 $\lambda x. M$，称之为 $x$ 在 $M$ 上的抽象
    2. 应用：对于表达式 $M$ 与 $N$，可构造新的表达式 $M \ N$，称之为 $M$ 对 $N$ 的应用
2. $\beta-$归约：形如 $(\lambda x.M) \ N$ 的表达式可被重写为 $M[x:=N]$，即 $M$ 中的任意 $x$ 都被替换为 $N$，将其称为 $(\lambda x.M) \ N$ 到 $M[x:=N]$ 的 $\beta-$规约
